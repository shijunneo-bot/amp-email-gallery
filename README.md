## Features

- **AMP Feasibility Panel** — every template rated Green / Amber / Red with infrastructure requirements
- **Mobile preview toggle** — simulates responsive rendering inside a phone frame
- **Full screen mode** — isolated preview per template
- **Copy pitch note** — one-click copy of client-ready feasibility notes for Amber/Red templates
- **Search + filter** — by brand, use case, feature, or vertical

## Usage

All templates are self-contained simulations built with vanilla JS. They demonstrate what AMP email interactions *look and feel like* — not live AMP running inside Gmail. For production AMP deployment, CleverTap's Email channel supports full AMP4Email with CORS endpoint integration.
